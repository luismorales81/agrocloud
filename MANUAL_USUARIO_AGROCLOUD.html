🌾 Bienvenido a AgroCloud

🚀 Funcionalidades Disponibles

🏠 Dashboard Principal

Vista general con métricas clave: número de campos, lotes activos, cultivos en producción, labores pendientes y valor del inventario. Incluye actividad reciente y estadísticas de labores.

🌾 Gestión de Campos

Crear, editar y gestionar campos agrícolas con integración de Google Maps para dibujar polígonos exactos. Visualización geográfica y cálculo automático de superficies.

🏞️ Gestión de Lotes

Administrar lotes dentro de los campos, asignar cultivos, registrar historial de acciones y seguimiento de rendimientos por lote específico.

🌱 Gestión de Cultivos

Catálogo completo de cultivos con información detallada, ciclos de vida, requerimientos y seguimiento de cultivos activos en cada lote.

🧪 Gestión de Insumos

Control de inventario de insumos (semillas, fertilizantes, agroquímicos), seguimiento de stock, alertas de stock bajo y valorización del inventario.

🔧 Gestión de Labores

Planificación y seguimiento de labores agrícolas (siembra, fertilización, cosecha), asignación de recursos, costos asociados y estado de ejecución.

🚜 Gestión de Maquinaria

Registro de maquinaria agrícola, costos por hora, consumo de combustible, mantenimientos programados y disponibilidad de equipos.

📊 Gestión de Rindes

Seguimiento de rendimientos por cultivo y lote, comparación con expectativas, análisis de productividad y registro de cosechas.

📈 Sistema de Reportes

Generación de reportes detallados: rendimientos por lote, producción por cultivo, análisis de costos y rentabilidad. Exportación a Excel.

👥 Gestión de Usuarios

Administración de usuarios del sistema, asignación de roles y permisos, control de acceso y auditoría de actividades.

⚙️ Configuración

Configuración de monedas (ARS/USD), cotizaciones automáticas, parámetros del sistema y personalización de la aplicación.

🌤️ Widget Meteorológico

Información meteorológica en tiempo real, pronósticos para la zona de trabajo y alertas climáticas relevantes para las actividades agrícolas.

🔧 Características Técnicas

📱 Diseño Responsive

Interfaz adaptativa que funciona perfectamente en computadoras, tablets y smartphones. Navegación optimizada para dispositivos móviles.

🌐 Funcionamiento Offline

Aplicación web progresiva (PWA) que permite trabajar sin conexión a internet. Sincronización automática cuando se restaura la conexión.

🗺️ Integración Google Maps

Herramientas avanzadas de mapeo para dibujar campos, visualizar lotes y obtener coordenadas precisas para cada área de trabajo.

💰 Conversión de Monedas

Sistema automático de conversión entre pesos argentinos y dólares estadounidenses con cotizaciones actualizadas en tiempo real.

🔒 Seguridad JWT

Autenticación segura con tokens JWT, sesiones protegidas y control de acceso basado en roles de usuario.

📊 Base de Datos Robusta

Almacenamiento seguro en PostgreSQL con respaldos automáticos y alta disponibilidad para garantizar la integridad de los datos.

💡 Consejos para una Mejor Experiencia

⚠️ Importante:
  • Utiliza un navegador moderno (Chrome, Firefox, Safari, Edge)
  • Mantén la conexión a internet para sincronización automática
  • Guarda tu trabajo regularmente
  • Revisa los reportes antes de tomar decisiones importantes
✅ Recomendaciones:
  • Explora todas las funcionalidades con diferentes roles de usuario
  • Prueba la funcionalidad offline desconectando internet
  • Utiliza el widget meteorológico para planificar actividades
  • Exporta reportes para análisis externos
  • Configura las monedas según tus necesidades
